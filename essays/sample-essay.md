This is a sample essay to demonstrate how your content will look with the Just The Docs theme. Replace this file with your own writing, or use it as a template.

## Getting Started

Each essay is a simple Markdown file placed in the `essays/` folder. The front matter at the top of the file controls the title that appears in navigation, the ordering, and which section it belongs to.

## Formatting Examples

You can use all standard Markdown formatting:

- **Bold text** for emphasis
- *Italic text* for softer emphasis
- `Inline code` for technical terms
- [Links](https://example.com) to external resources

> Blockquotes work well for highlighting key passages or attributing quotes to others.

### Subheadings

Use subheadings to break up longer essays into logical sections. The theme automatically generates a table of contents on the right side of the page for easy navigation within longer pieces.
